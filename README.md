## Graph Search Algorithms

### Requirements
* Python (>=3.5.*)

### How to
* Input (command line or file):
<<i>number of vertices</i>> 
<<i>adjacency matrix</i>> 

* Run: python main.py -<<i>option</i>> < <i>input file</i>>
	* <<i>option</i>>  can be: 
	* b for BFS 
	* d for Dijkstra
	* bf for Bellman-Ford
	* fw for Ford-Warshall
